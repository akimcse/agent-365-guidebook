데모 시나리오

Agent 365 위협 탐지를 계층별로 검증하기 위한 참조입니다. 각 계층(콘텐츠·행위·데이터·신원)에서 무엇이 · 어느 엔진으로 · 어느 포털에서 탐지되는지를 정리하고, Copilot Studio ×2 · Foundry ×2 커스텀 에이전트로 플랫폼 간 차이를 대조합니다.

SECTION 04

갭 · 확인 필요진행 전 점검

  1. 콘텐츠 층 XDR 알림은 Foundry 경로만 — CS Wiki는 내장 모더레이션 차단분이 Purview/CS 분석에만 기록(검증 완료).
  2. 행위 층은 tool 경계 도달 필요 — 에이전트가 tool 호출 전 거부하면 XDR·BehaviorInfo 모두 비어 있음.
  3. Risky Agent IRM alert 버그 — 신원 층 시연이 기대대로 안 될 수 있음.
