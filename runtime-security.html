런타임 위협 탐지

Copilot Studio / Azure AI Foundry 에이전트의 런타임 위협 탐지·차단은 서로 다른 3개 계층에서 일어납니다 — ① Copilot Studio 네이티브, ② Defender for AI Services(MDC, 모델 레벨), ③ Agent 365 · Defender XDR(에이전트 런타임). 계층마다 탐지 엔진 · 표출 포털 · 라이선스 · Azure 구독 필요 여부가 다릅니다.

SECTION 03

Defender 포털"Security for AI" 설정의 커넥터가 어떻게 엮여 합쳐지나

Defender 포털 System › Settings › Security for AI 화면은 두 그룹으로 나뉘며, 각 그룹에 에이전트 유형별 커넥터가 있습니다. 이 커넥터들이 위 ① 아키텍처의 posture(①의 미래/조사)와 real-time protection(레이어 A·B)을 각각 켜고, 마지막 M365 커넥터가 조사·Advanced Hunting을 묶어 모두 Defender 포털에서 합쳐집니다.

Microsoft Defender 포털 (XDR)

posture(인벤토리·attack path·exposure) + real-time protection(Incidents & Alerts) + M365 커넥터(조사·Advanced Hunting)가 한 포털로 수렴

🔑 M365 커넥터 = linchpin. 이게 연결돼야 RTP 탐지·차단의 alert·조사·Advanced Hunting이 Defender 포털에 모입니다. 커넥터를 켜면 Defender for Cloud Apps(MDA)의 ingestion·threat detection이 포함됩니다. 끊기면 ① 아키텍처대로 차단은 작동하지만 alert·조사는 포털에 미표출됩니다.

SECTION 05

Jailbreak · Prompt Injection 탐지 — 계층별 라이선스 전제3계층 탐지 모델의 라이선스 · Azure 구독 · 2026.7.1 이관 정리

계층 무엇이 탐지하나 Azure 구독(MDC) 필요? 필요 라이선스
① Copilot Studio 네이티브
UPIA·XPIA 인라인 차단 · Security analytics		 Copilot Studio 기본 내장 콘텐츠 보호("secure by default"). 메이커 Security analytics에 차단 통계 표출 아니오 Copilot Studio 테넌트 + User 라이선스 (Azure 전제 없음)
② Defender for AI Services (MDC)
Azure OpenAI · Foundry 모델 레벨		 Azure AI Content Safety Prompt Shields + Threat Intel. 알림 예: AI.Azure_Jailbreak.* · AI.Azure_ASCIISmuggling → Defender XDR 연동 예 — 필수 Azure 구독 + Defender for Cloud 플랜 활성화 + Owner/Contributor · 토큰당 과금(30일·750억 토큰 무료 체험)
③ Agent 365 / Defender XDR
에이전트 런타임 (Copilot Studio · Foundry 에이전트)		 Agent 365 observability 기반 Defender XDR. jailbreak·XPIA·악성 콘텐츠 전파·시크릿 유출 등 탐지, M365 커넥터 + Copilot Studio(PPAC) 연동 시 실시간 차단 아니오 Microsoft Agent 365 라이선스(사용자당 $15/mo 단독 또는 M365 E7 포함) · 전제: M365 E5(또는 Defender+Purview Suite)
⚠️ 2026.7.1 라이선스 이관 (검증됨) 2026년 7월 1일부터 Copilot Studio · Foundry 에이전트의 "에이전트 레벨" 위협 탐지·실시간 보호는 Defender for Cloud Apps/Defender for Cloud에서 빠져 Microsoft Agent 365 라이선스로 이관됩니다(해당 라이선스 없는 테넌트는 접근 상실). 단 모델 레벨(Azure OpenAI 등 Foundry Models)의 jailbreak 탐지는 Defender for AI Services 플랜(Azure 구독 기반)에 그대로 잔류합니다. 또한 이관일에 기존 Agent 365 규칙의 Block 동작이 중단되므로, 차단을 유지하려면 새 real-time protection 정책으로 규칙을 재정의해야 합니다.
🎯 모델 레벨 vs 에이전트 레벨 2026.7.1 이후 에이전트 레벨 탐지(Copilot Studio·Foundry 에이전트)는 Agent 365 라이선스로, 모델 레벨(Azure OpenAI 등 Foundry Models) jailbreak 탐지는 Defender for AI Services(Azure 구독)로 이원화됩니다. 경계가 문서상 정밀히 그어져 있지 않으므로, 커버 범위 설계 시 두 축을 함께 고려해야 합니다.
