# Microsoft Agent 365 — 에이전트를 위한 컨트롤 플레인

> Agent 365로 AI 에이전트를 안심하고 관찰하며, 관리하고, 보호하세요. Microsoft 365 및 Microsoft Security 제어 기능을 확장해 대규모 에이전트형 AI를 효과적으로 관리하세요.

이 가이드북은 [공식 Microsoft Agent 365 페이지](https://www.microsoft.com/ko-kr/microsoft-agent-365)의 구성과 내용을 반영한 홈과, 세 개의 심화 서브페이지(에이전트 유형 분류·신원/위험·런타임 위협 탐지)로 구성됩니다.

## 개요 (Overview)

Agent 365는 AI 에이전트를 **관찰(Observe)** 하고, **관리·거버넌스(Govern)** 하며, **보호(Secure)** 하기 위한 컨트롤 플레인입니다. 조직은 에이전트 요구 사항에 맞게 맞춤화된 용도로 빌드된 기능을 사용하여, 사용자를 위한 기존 인프라를 에이전트로 확장할 수 있습니다.

Agent 365는 에이전트 관리를 **네 가지 핵심 관리자 역할**에 맞춰 최적화합니다.

- **Microsoft Defender 관리자** — 포괄적인 보안 태세와 고급 위협 방지를 에이전트까지 확장합니다.
- **Microsoft Entra 관리자** — 에이전트 ID를 보호하고 앱·리소스·인터넷·다른 에이전트에 대한 액세스를 안전하게 관리합니다.
- **Microsoft Purview 관리자** — 조직 전반에서 에이전트가 사용·생성하는 데이터를 관리·보호합니다.
- **Microsoft 365 관리자** — 조직 내 모든 에이전트를 종합적으로 파악하고, IT가 제어하는 워크플로로 온보딩하며, 보안 정책 템플릿을 적용해 모든 에이전트가 안전·규정 준수 상태로 시작하도록 하고, 에이전트 분석을 실시간으로 추적합니다.

## 관찰 (Observe)

Microsoft 365 채널을 통해 게시되고 **Entra 에이전트 ID로 등록된 모든 에이전트는 Agent 365 인벤토리에 자동으로 표시**됩니다. 이러한 환경의 외부에서 빌드된 에이전트에는 추가 단계가 필요합니다.

## 거버넌스 (Govern)

Agent 365는 에이전트와, 에이전트를 만들거나 관리하는 사용자 모두에 대해 **검색·수명 주기 관리·IT 정의 가드레일**을 포함한 포괄적인 에이전트 관리를 제공합니다.

## 보안 (Secure)

Agent 365는 **최소 권한 액세스**를 적용하고, 리소스에 대한 에이전트 액세스를 보호하고, 중요한 데이터를 보호하며, **고급 위협 방지 및 취약성 관리**를 포함합니다. 이러한 제어는 Microsoft Entra, Microsoft Purview, Microsoft Defender 관리 센터에서 보안 전문가가 직접 운영합니다.

## 라이선스

Agent 365는 **사용자 단위**로 라이선스가 부여됩니다. Agent 365에서 관리되는 에이전트와 상호 작용하거나 이를 소유·관리·후원하는 모든 사용자에게 권장됩니다. Agent 365는 **Microsoft 365 E7(The Frontier Suite)** 에 기본 포함됩니다.
