# Microsoft Agent 365 — 에이전트를 위한 컨트롤 플레인 (홈 텍스트본)

> 에이전트가 일상 업무의 일부가 되어도 완벽한 통제권을 유지하세요.
> Agent 365로 AI 에이전트를 안심하고 관찰하며, 관리하고, 보호하세요. Microsoft 365 및 Microsoft Security 제어 기능을 확장해 대규모 에이전트형 AI를 효과적으로 관리하세요.

## 개요

Agent 365는 AI 에이전트를 **관찰(Observe)** 하고, **관리·거버넌스(Govern)** 하며, **보호(Secure)** 하기 위한 컨트롤 플레인입니다. 조직은 용도에 맞게 빌드된 기능으로 기존 인프라를 에이전트로 확장할 수 있습니다. Agent 365는 에이전트 관리를 네 가지 핵심 관리자 역할에 맞춰 최적화합니다.

- **Microsoft Defender 관리자** — 포괄적인 보안 태세 및 고급 위협 방지를 에이전트까지 확장합니다.
- **Microsoft Entra 관리자** — 에이전트 ID를 보호하고 앱·리소스·인터넷·다른 에이전트에 대한 액세스를 안전하게 관리합니다.
- **Microsoft Purview 관리자** — 조직 전반에서 에이전트가 사용·생성하는 데이터를 관리·보호합니다.
- **Microsoft 365 관리자** — 모든 에이전트를 종합 파악하고 IT 제어 워크플로로 온보딩하며, 보안 정책 템플릿 적용·실시간 에이전트 분석을 수행합니다.

## 핵심 기능

### 관찰 가능성 (Observability)
- **레지스트리** — Microsoft AI 플랫폼 기반 에이전트, 에코시스템 파트너 에이전트, 직접 등록한 에이전트를 포함해 조직 내 모든 에이전트를 완벽하게 파악.
- **에이전트 맵** — 에이전트 활동 및 연결을 시각적으로 매핑.
- **에이전트 분석** — 에이전트 사용 인사이트를 실시간 추적.
- **역할별 감독** — 각 관리자 역할에 맞춰 에이전트를 감독.

### 거버넌스 (Governance)
- **에이전트 온보딩** — IT 제어 워크플로로 온보딩 + 보안 정책 템플릿 적용.
- **통합 관리** — 사용자·데이터·도구·MCP 서버를 제어해 최소 권한 액세스 적용.
- **수명 주기 관리** — 비활성 만료, 소유자 없는 에이전트 표시, 위험 에이전트 차단 등 자동 적용.
- **감사 및 로깅** — 로그 기록·보고·감사로 가시성·추적성 강화.
- **데이터 규정 준수** — 비윤리적 상호 작용을 감지·보존·조사하는 콘텐츠 안전 제어.

### 보안 (Security)
- **액세스 제어** — 조건부 액세스·인터넷 트래픽 필터링을 사용자에서 에이전트까지 확장.
- **데이터 보안** — AI 데이터 노출 현황 파악 + 과도한 공유·유출·위험 사용으로부터 보호.
- **위협 방지** — 위협·취약점·적대적 공격으로부터 보호, 신속한 탐지·조사·대응.

## 익숙한 관리 도구로 함께 관리

Microsoft 365 관리 센터 · Microsoft Defender · Microsoft Entra · Microsoft Purview

## Microsoft 365 E7

Agent 365는 **Microsoft 365 E7(The Frontier Suite)** 에 기본 포함됩니다. E7은 Copilot, AI 에이전트, 엔터프라이즈급 보안·거버넌스·관리를 통합하며, Agent 365는 그 안의 컨트롤 플레인으로 중앙 집중식 거버넌스·보안·가시성을 제공합니다.

## 라이선스

Agent 365는 **사용자 단위**로 라이선스가 부여됩니다. 관리되는 에이전트와 상호 작용하거나 이를 소유·관리·후원하는 모든 사용자에게 권장됩니다.

## FAQ 요약

1. **Agent 365란?** 에이전트를 관찰·관리·보호하는 컨트롤 플레인(4개 관리자 역할).
2. **작동 에이전트 유형?** M365 채널로 게시 + Entra 에이전트 ID로 등록된 에이전트는 자동 인벤토리 표시; 외부 빌드는 추가 단계 필요.
3. **관리 도움?** 검색·수명 주기 관리·IT 정의 가드레일 포함 포괄적 관리.
4. **보안·데이터 보호?** 최소 권한 + 데이터 보호 + 고급 위협 방지·취약성 관리(Entra/Purview/Defender).
5. **라이선스?** 사용자 단위, M365 E7에 기본 포함.
6. **E7 포함?** 네, M365 E7(The Frontier Suite)에 포함.
