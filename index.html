Microsoft Agent 365

에이전트를 위한 컨트롤 플레인

Agent 365로 AI 에이전트를 안심하고 관찰하며, 관리하고, 보호하세요. Microsoft 365 및 Microsoft Security 제어 기능을 확장해 대규모 에이전트형 AI를 효과적으로 관리하세요.

Agent 365는 AI 에이전트를 관찰(Observe)하고, 관리·거버넌스(Govern)하며, 보호(Secure)하기 위한 컨트롤 플레인입니다. 조직은 용도에 맞게 빌드된 기능으로 기존 인프라를 에이전트로 확장할 수 있습니다. Agent 365는 에이전트 관리를 네 가지 핵심 관리자 역할에 맞춰 최적화합니다.

Microsoft Defender 관리자

포괄적인 보안 태세와 고급 위협 방지를 에이전트까지 확장합니다.

Microsoft Entra 관리자

에이전트 ID를 보호하고 앱·리소스·인터넷·다른 에이전트에 대한 액세스를 안전하게 관리합니다.

Microsoft Purview 관리자

조직 전반에서 에이전트가 사용·생성하는 데이터를 관리·보호합니다.

Microsoft 365 관리자

모든 에이전트를 종합 파악하고 IT 워크플로로 온보딩하며, 보안 정책 템플릿 적용·실시간 분석을 수행합니다.

Agent 365 자세히 보기 공식 페이지

관찰(Observe) — Microsoft 365 채널을 통해 게시되고 Entra 에이전트 ID로 등록된 모든 에이전트는 Agent 365 인벤토리에 자동으로 표시됩니다. 이러한 환경의 외부에서 빌드된 에이전트에는 추가 단계가 필요합니다.

👉 심화: ② 에이전트 신원 · 위험에서 어떤 신원으로 인증하고 어떤 로그에 표시되는지 다룹니다.

거버넌스(Govern) — Agent 365는 에이전트와, 에이전트를 만들거나 관리하는 사용자 모두에 대해 검색·수명 주기 관리·IT 정의 가드레일을 포함한 포괄적인 관리를 제공합니다.

👉 심화: ① 에이전트 유형 분류에서 실행 위치·역할·제작 방식별 거버넌스 통제 지점을 다룹니다.

보안(Secure) — Agent 365는 최소 권한 액세스를 적용하고, 리소스에 대한 에이전트 액세스를 보호하며, 중요한 데이터를 보호하고, 고급 위협 방지 및 취약성 관리를 포함합니다. 이러한 제어는 Microsoft Entra·Purview·Defender 관리 센터에서 보안 전문가가 직접 운영합니다.

👉 심화: ③ 런타임 위협 탐지에서 콘텐츠/액션 2-레이어 탐지 아키텍처를 다룹니다.

라이선스 — Agent 365는 사용자 단위로 라이선스가 부여됩니다. 관리되는 에이전트와 상호 작용하거나 이를 소유·관리·후원하는 모든 사용자에게 권장됩니다. Agent 365는 Microsoft 365 E7(The Frontier Suite)에 기본 포함됩니다.
