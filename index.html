✦ Microsoft Agent 365

에이전트가 일상 업무의 일부가 되어도 완벽한 통제권을 유지하세요.

Agent 365로 AI 에이전트를 안심하고 관찰하며, 관리하고, 보호하세요. Microsoft 365 및 Microsoft Security 제어 기능을 확장해 대규모 에이전트형 AI를 효과적으로 관리하세요.

공식 페이지

핵심 기능

관찰 가능성 · 거버넌스 · 보안의 세 가지 축으로 에이전트를 완전히 통제합니다.

관찰 가능성

Observability

조직 내 모든 에이전트를 완벽하게 파악

레지스트리

Microsoft AI 플랫폼을 기반으로 구축된 에이전트, 에코시스템 파트너 에이전트, 직접 등록한 에이전트를 포함하여 조직 내 모든 에이전트를 완벽하게 파악할 수 있습니다.

에이전트 맵

에이전트 활동 및 연결을 시각적으로 매핑하여 상호 작용 관계를 한눈에 파악합니다.

에이전트 분석

에이전트 사용 인사이트를 실시간으로 추적하여 활용 현황을 측정합니다.

역할별 감독

각 관리자 역할에 맞춰 에이전트를 감독하고 책임 범위를 명확히 합니다.

거버넌스

Governance

검색·수명 주기 관리·IT 정의 가드레일

에이전트 온보딩

IT가 제어하는 워크플로를 통해 에이전트를 온보딩하고 보안 정책 템플릿을 적용하여, 모든 에이전트가 안전하고 관리되며 규정을 준수하는 상태로 시작하도록 합니다.

통합 관리

에이전트가 사용할 수 있는 사용자, 데이터, 도구, MCP 서버를 제어하여 최소 권한 액세스를 적용합니다. 필요한 리소스와 다른 에이전트에만 액세스하도록 제한합니다.

수명 주기 관리

규칙 기반 관리를 통해 비활성 에이전트 만료, 소유자 없는 에이전트 표시, 위험한 에이전트 차단 등 수명 주기 정책을 자동 적용합니다.

감사 및 로깅

로그 기록·보고·감사 기능으로 에이전트의 활동과 상호 작용에 대한 가시성과 추적성을 강화합니다.

데이터 규정 준수

비윤리적 에이전트 상호 작용을 감지·보존·조사할 수 있는 콘텐츠 안전 제어를 구축합니다.

보안

Security

최소 권한 · 데이터 보호 · 고급 위협 방지

액세스 제어

조건부 액세스와 인터넷 트래픽 필터링 정책을 사용자에서 에이전트까지 확장 적용하여 에이전트 ID를 보호하고 보안 침해를 방지합니다.

데이터 보안

AI 관련 데이터 노출 현황을 파악하고, 에이전트가 생성·활용하는 데이터를 과도한 공유·유출·위험한 사용 행위로부터 안전하게 보호합니다.

위협 방지

에이전트를 위협·취약점·적대적 공격으로부터 보호하고, 공격 경로를 명확히 파악하여 인시던트를 신속하게 탐지·조사·대응합니다.

FAQ

Microsoft Agent 365란?
Agent 365는 AI 에이전트를 관찰·관리·보호하기 위한 컨트롤 플레인입니다. 에이전트 관리를 네 가지 핵심 역할에 맞춰 최적화합니다.
  • Microsoft Defender 관리자 — 포괄적인 보안 태세 및 고급 위협 방지를 에이전트까지 확장합니다.
  • Microsoft Entra 관리자 — 에이전트 ID를 보호하고 앱·리소스·인터넷·다른 에이전트에 대한 액세스를 안전하게 관리합니다.
  • Microsoft Purview 관리자 — 조직 전반에서 에이전트가 사용·생성하는 데이터를 관리·보호·관리합니다.
  • Microsoft 365 관리자 — 모든 에이전트를 종합 파악하고 IT 제어 워크플로로 온보딩하며, 보안 정책 템플릿 적용·실시간 분석을 수행합니다.
Agent 365에서 작동하는 에이전트 유형은 무엇인가요?
Microsoft 365 채널을 통해 게시되고 Entra 에이전트 ID로 등록된 모든 에이전트는 Agent 365 인벤토리에 자동으로 표시됩니다. 이러한 환경의 외부에서 빌드된 에이전트에는 추가 단계가 필요합니다.
Agent 365는 조직에서 AI 에이전트를 관리하는 데 어떻게 도움이 됩니까?
Agent 365는 에이전트와, 에이전트를 만들거나 관리하는 사용자 모두에 대해 검색·수명 주기 관리·IT 정의 가드레일을 비롯한 포괄적인 에이전트 관리를 제공합니다.
Agent 365는 에이전트에 대한 보안 및 데이터 보호를 어떻게 제공하나요?
Agent 365는 최소 권한 액세스를 적용하고, 리소스에 대한 에이전트 액세스를 보호하고, 중요한 데이터를 보호하며, 고급 위협 방지 및 취약성 관리를 포함합니다. 이러한 제어는 Microsoft Entra·Purview·Defender 관리 센터에서 보안 전문가가 직접 운영합니다.
Agent 365는 어떤 방식으로 라이선스가 제공되나요?
Agent 365는 사용자 단위로 라이선스가 부여됩니다. 관리되는 에이전트와 상호 작용하거나 이를 소유·관리·후원하는 모든 사용자에게 권장됩니다. Agent 365는 Microsoft 365 E7에 기본 포함됩니다.
Agent 365가 Microsoft 365 E7에 포함되어 있나요?
네, Microsoft 365 E7(The Frontier Suite) 라이선스에는 Agent 365가 포함됩니다. Frontier 제품군은 Copilot, AI 에이전트, 엔터프라이즈급 보안·거버넌스·관리를 통합하며, Agent 365는 ME7 내의 컨트롤 플레인으로 중앙 집중식 거버넌스·보안·가시성을 제공합니다.
